    Smartsheet TQ/RFI Register#

    Column Summary#

    Title Brief Description
    Detailed Status The detailed status of an RTI which shows precisely where it is within the workflow.
    Detailed Status (TQ) Determines the detailed status for technical queries.
    Detailed Status (RFI) Determines the detailed status for requests for information.
    Exceeds Max Response Days Determines the detailed status for requests for information.
    Overdue (or Late) Indicates whether a response is overdue, or was provided later than required/requested.
    Status A high-level status for RTIs for client reporting purposes.

    Column Details#

    Detailed Status#

    Provides the detailed, system level status of the RTI as either: Preparing, Issued, Re-Issued, or Closed. (An indeterminate status of Unknown Detailed Status is also possible if an error occurs.)

    The detailed status is the non-client-facing status for internal reports and for determining the precise status of RTIs in the workflow.

    Formula

    =IF(
    [Detailed Status (TQ)]@row <> "", 
    [Detailed Status (TQ)]@row, 
    [Detailed Status (RFI)]@row
)

    Detailed Status (TQ)#

    Provides the detailed, system level status for TQs only, as one of the following:

    • Approver Actioning
    • Approving
    • Assessing Response
    • Authoriser Actioning
    • Authorising
    • Cancelled
    • Client Actioning
    • Client Finalising Response
    • Client Reviewing SME Response
    • Closed
    • Further Requested
    • Issued
    • Revising
    • SME Actioning
    • An indeterminate status of Unknown Detailed Status is also possible if an error occurs.

    The detailed status for the TQ is used as the Detailed Status where applicable. (TQ and RFI formulas are only split due to Smartsheet size constraints.)

    Formula

    =IF(
    [Type of Request]@row = "Technical Query (TQ)", 
    IF(
        [Status: Technical Approval (by GTE Tech Approver)]@row = "Rejected", 
        IF(
            [Workflow: Approver's Direction]@row = "Cancel Request", 
            "Cancelled", 
            IF(
                [Workflow: Approver's Direction]@row = "Request Revision (by Originator)", 
                IF(
                    [Workflow: Originator's Direction after Revision]@row = "Submit Revision", 
                    "Approving", 
                    IF(
                        [Workflow: Originator's Direction after Revision]@row = "Cancel Request", 
                        "Cancelled",
                        "Revising"
                    )
                ), 
                IF(
                    [Workflow: Approver's Direction]@row = "Approve (and Request Authorisation)", 
                    "Authorising", 
                    "Approver Actioning"
                )
            )
        ), 
        IF(
            [Status: Technical Approval (by GTE Tech Approver)]@row = "Approved", 
            IF(
                [Status: Authorisation (by GTE PM)]@row = "Rejected", 
                IF(
                    [Workflow: Authoriser's Direction for TQ]@row = "Cancel Request", 
                    "Cancelled", 
                    IF(
                        [Workflow: Authoriser's Direction for TQ]@row = "Request Revision (by Originator)", 
                        "Revising", 
                        IF(
                            [Workflow: Authoriser's Direction for TQ]@row = "Request Approval for Amendments", 
                            "Approving", 
                            IF(
                                [Workflow: Authoriser's Direction for TQ]@row = "Authorise (and Issue)", 
                                "Issued", 
                                "Authoriser Actioning"
                            )
                        )
                    )
                ), 
                IF(
                    [Status: Authorisation (by GTE PM)]@row = "Authorised", 
                    IF(
                        [Status: Recommendation Approval (by Client)]@row = "Approved as Recommended", 
                        IF(
                            [Status: Acceptance of Client Response (by GTE PM)]@row = "Accepted", 
                            "Closed", 
                            IF(
                                [Status: Acceptance of Client Response (by GTE PM)]@row = "Rejected", 
                                IF(
                                    [Workflow: Means to Address]@row = "Delegate for Response (to SME Selected)", 
                                    IF(
                                        [Status: Acceptance of SME Response (by Client)]@row = "Accepted", 
                                        "Client Finalising Response", 
                                        IF(
                                            [Status: Acceptance of SME Response (by Client)]@row = "Rejected", 
                                            "SME Actioning", 
                                            "Unknown Detailed Status"
                                        )
                                    ), 
                                    "Further Requested"
                                ), 
                                IF(
                                    [Workflow: Means to Address]@row = "Submit Response (As Revised)", 
                                    "Assessing Response", 
                                    IF(
                                        [Workflow: Means to Address]@row = "Select Action", 
                                        "Further Requested", 
                                        "Assessing Response"
                                    )
                                )
                            )
                        ), 
                        IF(
                            [Status: Recommendation Approval (by Client)]@row = "Not Approved as Recommended", 
                            IF(
                                [Response (Complete When Ready to Issue to GTE)]@row <> "", 
                                IF(
                                    [Status: Acceptance of Client Response (by GTE PM)]@row = "Accepted", 
                                    "Closed", 
                                    IF(
                                        [Status: Acceptance of Client Response (by GTE PM)]@row = "Rejected", 
                                        IF(
                                            [Workflow: Means to Address]@row = "Delegate for Response (to SME Selected)", 
                                            IF(
                                                [Status: Acceptance of SME Response (by Client)]@row = "Accepted", 
                                                "Client Finalising Response", 
                                                IF(
                                                    [Status: Acceptance of SME Response (by Client)]@row = "Rejected", 
                                                    "SME Actioning", 
                                                    "Unknown Detailed Status"
                                                )
                                            ), 
                                            "Further Requested"
                                        ), 
                                        IF(
                                            [Workflow: Means to Address]@row = "Submit Response (As Revised)", 
                                            "Assessing Response", 
                                            "Unknown Detailed Status"
                                        )
                                    )
                                ), 
                                IF(
                                    [Nominated Delegate or SME (if Req.)]@row <> "", 
                                    IF(
                                        [Technical Response]@row <> "", 
                                        IF(
                                            [Status: Acceptance of SME Response (by Client)]@row = "Accepted", 
                                            "Client Finalising Response", 
                                            IF(
                                                [Status: Acceptance of SME Response (by Client)]@row = "Rejected", 
                                                "SME Actioning", 
                                                "Client Reviewing SME Response"
                                            )
                                        ), 
                                        "SME Actioning"
                                    ), 
                                    "Client Actioning"
                                )
                            ), 
                            "Issued"
                        )
                    ), 
                    "Authorising"
                )
            ), 
            "Approving"
        )
    )
)

    Detailed Status (RFI)#

    Provides the detailed, system level status for RFIs only, as one of the following:

    • Assessing Response
    • Authoriser Actioning
    • Authorising
    • Cancelled
    • Client Actioning
    • Client Finalising Response
    • Client Reviewing SME Response
    • Closed
    • Further Requested
    • Issued
    • Revising
    • SME Actioning

    The detailed status for the RFI is used as the Detailed Status where applicable. (TQ and RFI formulas are only split due to Smartsheet size constraints.)

    Formula

    =IF(
    [Type of Request]@row = "Request for Information (RFI)", 
    IF(
        [Status: Acceptance of Client Response (by GTE PM)]@row <> "Rejected", 
        IF(
            [Status: Authorisation (by GTE PM)]@row = "Authorised", 
            IF(
                [Response (Complete When Ready to Issue to GTE)]@row <> "", 
                IF(
                    [Status: Acceptance of Client Response (by GTE PM)]@row = "Accepted", 
                    "Closed", 
                    IF(
                        [Status: Acceptance of Client Response (by GTE PM)]@row = "Rejected", 
                        "Further Requested", "Assessing Response"
                    )
                ), 
                IF(
                    [Nominated Delegate or SME (if Req.)]@row <> "", 
                    IF(
                        [Technical Response]@row <> "", 
                        IF(
                            [Status: Acceptance of SME Response (by Client)]@row = "Accepted", 
                            "Client Finalising Response", 
                            IF(
                                [Status: Acceptance of SME Response (by Client)]@row = "Rejected", 
                                "SME Actioning", 
                                "Client Reviewing SME Response"
                            )
                        ), 
                        "SME Actioning"
                    ), 
                    "Client Actioning"
                )
            ), 
            IF(
                [Status: Authorisation (by GTE PM)]@row = "Rejected", 
                IF(
                    [Workflow: Authoriser's Direction for RFI]@row = "Cancel Request", 
                    "Cancelled", 
                    IF(
                        [Workflow: Authoriser's Direction for RFI]@row = "Request Revision (by Originator)", 
                        IF(
                            [Workflow: Originator's Direction after Revision]@row = "Submit Revision", 
                            "Authorising", 
                            IF(
                                [Workflow: Originator's Direction after Revision]@row = "Cancel Request", 
                                "Cancelled", 
                                "Revising"
                            )
                        ), 
                        IF(
                            [Workflow: Authoriser's Direction for RFI]@row = "Authorise (and Issue)", 
                            "Issued", 
                            "Authoriser Actioning"
                        )
                    )
                ), 
                "Authorising"
            )
        ), 
        IF(
            [Workflow: Means to Address]@row = "Delegate for Response (to SME Selected)", 
            IF(
                [Status: Acceptance of SME Response (by Client)]@row = "Accepted", 
                "Client Finalising Response", 
                "SME Actioning"
            ), 
            IF(
                [Status: Acceptance of SME Response (by Client)]@row = "Rejected", 
                "SME Actioning", 
                IF(
                    [Workflow: Means to Address]@row = "Submit Response (as Revised)", 
                    "Assessing Response", 
                    "Further Requested"
                )
            )
        )
    )
)

    Exceeds Max Response Days#

    Indicates when a response exceeded the maximum response time, as established in GTE's proposal (nominally 10 days).

    Formula

    =AND(
    [Maximum Response Days]# <> "", 
    IF(
        [Responded Date]@row <> "", 
        [Responded Date]@row - [Issued Date]@row > [Maximum Response Days]#, 
        TODAY() - [Issued Date]@row > [Maximum Response Days]#
    )
)

    Overdue (or Late)#

    Indicates when a response is overdue, or was provided later than required/requested. Where a response has not been received, this is because the current date exceeds the Response Required by Date. If a response has been received then the Responded Date must be greater than the Response Required by Date.

    Formula

    =AND(
    [Response Required by Date]@row <> "", 
    IF(
        [Responded Date]@row <> "", 
        [Responded Date]@row > [Response Required by Date]@row, 
        TODAY() > [Response Required by Date]@row
    )
)

    Status#

    Provides the high level status of the RTI as either:

    • Preparing
    • Issued
    • Re-Issued
    • Responded
    • Closed
    • An indeterminate status of Unknown Status is also possible if an error occurs.

    The high level status is the client-facing reporting status.

    Formula

    =IF(
    OR(
        [Detailed Status]@row = "Approving", 
        [Detailed Status]@row = "Authorising", 
        [Detailed Status]@row = "Authoriser Actioning", 
        [Detailed Status]@row = "Approver Actioning", 
        [Detailed Status]@row = "Revising"
    ), 
    "Preparing", 
    IF(
        OR(
            [Detailed Status]@row = "Issued", 
            [Detailed Status]@row = "Client Actioning", 
            [Detailed Status]@row = "SME Actioning", 
            [Detailed Status]@row = "Client Reviewing SME Response", 
            [Detailed Status]@row = "Client Finalising Response"
        ),
        IF(
            [Overdue (or Late)]@row = 0,
            "Issued",
            "Overdue"
        ), 
        IF(
            [Detailed Status]@row = "Further Requested",
            IF(
                [Overdue (or Late)]@row = 0,
                "Re-Issued",
                "Overdue"
            ), 
            IF(
                OR(
                    [Detailed Status]@row = "Cancelled", 
                    [Detailed Status]@row = "Closed"
                ),
                "Closed",
                IF(
                    [Detailed Status]@row = "Assessing Response", 
                    "Responded", 
                    "Unknown Status"
                )
            )
        )
    )
)

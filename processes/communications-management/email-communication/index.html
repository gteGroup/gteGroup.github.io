    Email Correspondence#

    To facilitate quick access to all project-related emails and thus allow to lookup any information pertinent to the project and communicated via emails - either internally or with any external project stakeholder - in a fast and effective way, GTE stores all projec related emails centrally in Project Project Folders. There is a set of rules whcih are followed in this respect and technical tools which are used in this process. For further details see section Project Public Folders of this documentation.
